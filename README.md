## 🛠 Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript  
- **Hosting**: Netlify (with free SSL)  
- **Design**: Custom UI with CSS animations  
- **Tools**: Git, GitHub, Font Awesome
